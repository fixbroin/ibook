# **App Name**: BroBookMe

## Core Features:

- Booking Link Generation: Generates a unique, shareable link for each provider to direct customers to their booking page.
- Automated Slot Generation: Automatically generates available time slots based on the provider's set working hours, slot duration, and booking delay preferences.
- Booking Confirmation and Notifications: Sends automated email notifications to both the customer and the provider upon successful booking, including booking details. Generative AI can be incorporated by acting as a tool, to make automated decisions to include and personalize additional service offering or products, relevant for this booking.
- Service Type Selection: Allows customers to select the service type (Online, Shop Visit, Doorstep) and fill in the required details accordingly. Each type may trigger completely different form flow.
- Provider Profile and Settings: Enables providers to configure their profile details, working hours, slot duration, service types, and booking delay preferences. The profile should contains business description.
- Bookings Management: Provides providers with a dashboard to view upcoming and past bookings.

## Style Guidelines:

- Primary color: Light grayish-blue (#B0C4DE), offering a professional and trustworthy feel. Convert from HSL(210, 30%, 75%)
- Background color: Very light gray (#F5F5F5) provides a clean and modern backdrop. Convert from HSL(0, 0%, 96%)
- Accent color: A vibrant light purple (#BFA0DE) used for calls to action and important highlights. Convert from HSL(283, 30%, 75%)
- Body and headline font: 'PT Sans' (sans-serif) provides a clean, modern look suitable for both headlines and body text.
- Simple, clear icons to represent different service types and booking options.
- Clean, intuitive layout with a focus on ease of use for both providers and customers.
- Subtle transitions and animations to enhance user experience and provide feedback on interactions.
